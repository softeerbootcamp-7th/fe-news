# fe-news

STEP1. 결과물 올릴 깃헙 연동하기(완료)
루카스 화면에서 원격 저장소에 나의 브랜치를 생성한다. (루카스 브랜치 생성 버튼)

이후 원격 저장소에 접속해서 branch 생성이 됐는지 확인한다.

원격 저장소에 접속해서 fork 한다.

fork한 나의 저장소에서 코드를 내려받고(pull) 개발할 준비를 한다.

STEP2. 개발환경 구성
NPM 환경을 검색하면서 이해하고 NPM 프로젝트 환경을 구성한다.

ESM 표준방식을 이해하고, 브라우저 화면에 hello world를 출력한다.

로컬 서버를 띄우는 방법을 찾아서 적용한다.

STEP3. 태스크 도출
뉴스스탠드 미션을 살펴보고, 이번주 개발해야 할 기능을 도출해서 readme 마크다운 문서에 정리해본다.(체크리스트 형태로).

참고로 개발 태스크는 작을수록 좋다.

마크다운 문서를 커밋하고, push 한 뒤에 원격저장소(upstream) 저본인의 branch로 PR을 보낸다.

STEP4. 첫 개발과 PR
첫페이지의 HTML 구조(CSS생략)만 1차로 개발하고 PR을 보낸다.

STEP5. 동료 작업 살펴보기
임의의 동료 PR을 확인해서 , 코드리뷰를 남겨본다.
