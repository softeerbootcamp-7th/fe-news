# 뉴스테스크 1주차 개발 기능 도출

## 1주차 개발 기능 도출

- 상단 영역
  - [x] 왼쪽 뉴스 스탠드 로고 클릭 시 화면 새로고침
  - [x] 오른쪽에 날짜 표시
- 최신 뉴스 자동 롤링 영역
  - [x] 헤드라인 5개가 5초마다 자동으로 무한 롤링.
  - [x] 좌우 영역의 시간차는 1초를 둔다
  - [x] 헤드라인 영역에 hover 시
    - [x] 마우스 커서는 손 모양
    - [x] 무한 롤링 일시정지, 헤드라인에 밑줄 표시
    - [x] 헤드라인 hover 종료 시 다시 무한 롤링
- 언론사별 기사들 확인 영역
  - [x] 그리드보기, 리스트 보기 2가지 옵션 아이콘
  - 전체 언론사 탭
    1. 그리드 보기
       - [x] 가로 930px, 세로 388px
       - [x] 6\*4 테이블 모양
       - [x] 언론사 브랜드 마크 정중앙에 배치
       - [ ] 페이지 새로고침 될 때 마다 브랜드 랜덤으로 재배치
       - 브랜드 영역 hover시
         - [x] 구독 하고 있지 않은 언론사라면 ‘구독하기’버튼 보인다
           - [ ] 구독하기 버튼 누르면 서버 처리 시간동안 버튼에 로딩 indicator 뜬다
           - [ ] 구독 완료 시 사용자가 구독한 언론사 탭의 리스트보기 화면으로 이동
         - [x] 이미 구독하고 있는 언론사의 경우 ‘해지하기’ 버튼 보인다
           - [ ] ‘{언론사 이름}을 구독해지하시겠습니까?’ 라는 알림창이 뜬다
             - 예 해지합니다
               - [ ] 즉시 구독 해지
               - [ ] 목록의 다음 순서 언론사 바로 나타난다
             - 아니오
               이 2가지 선택지 버튼
           - [ ] 각 선택지 버튼에 마우스 올리면 텍스트 밑줄
       - [x] 그리드 좌우에 페이지 넘길 수 있는 화살표 배치
       - [ ] 가장 첫 페이지, 가장 끝 페이지에서 나올 필요 없는 화살표는 아예 표시되지 않도록 없앤다
       - [x] 최대 4페이지까지만
  - 내가 구독한 언론사 탭
    - [x] 내가 구독한 언론사 숫자 표시
    1. 그리드 보기
       - [x] 사용자가 구독중인 언론사의 브랜드 마크만 보이고 나머지 그리드 칸은 비어있음
       - [ ] 전체 언론사의 그리드 보기 화면과 동일 (언론사 브랜드 hover시 동작만 다름)
       - [ ] 구독중인 언론사 브랜드 마크 셀에 마우스 올리면 ‘구독해지’ 버튼 → 누르면 알림창
         - [ ] 구독 해지 즉시 그리드에서 해당 언론사 브랜드 마크 사라짐
