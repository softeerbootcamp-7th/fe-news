## 2주 개발 리스트

### 1. Grid ↔ List 전환 구조 설계
- 뉴스 스탠드 UI를 Grid / List 뷰로 전환할 수 있도록 설계
- 탭 클릭 시 기존 DOM을 유지하지 않고 container.innerHTML = '' 기반으로 뷰를 완전히 교체하는 방식 채택
- 상태는 subscriptionStore에서 관리하고, 렌더링은 view 단위로 분리하여 DOM 상태 의존성 제거

### 2. Grid 렌더링 로직 점검 및 디버깅
- renderGrid 함수에서 DOM이 제거되지 않는 이슈 확인
- 원인이 selector 오타임을 확인하고 수정
- innerHTML = '' 방식이 정상적으로 동작함을 재확인

### 3. 데이터 가공: category 중복 제거 로직 구현
- 언론사 데이터에서 category 값만 추출
- Set을 활용해 중복 없는 category 목록 생성
- 카테고리 탭 / 필터 UI에 활용 가능한 형태로 정리
```js
const categories = [...new Set(
  presses.map(press => press.category)
)];
```
