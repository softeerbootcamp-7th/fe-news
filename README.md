# fe-news

## 태스크 정리

### 1. 기본 상단 영역
- [ ] 로고, 날짜 표시 레이아웃
- [ ] 로고 클릭 시 화면 새로고침 기능

### 2. 최신 뉴스 자동롤링 영역
- [ ] 뉴스 카드 레이아웃
- [ ] 5초마다 뉴스 전환 기능 (좌우영역 시간차 1초)
- [ ] 호버 시 해당 영역만 롤링 일시정지, 헤드라인 밑줄 표시, 마우스커서 모양 변경
- [ ] 호버 제거 시 좌우영역 롤링 시간차 1초 유지
- [ ] 뉴스 제목 롤링 애니메이션
- [ ] 뉴스 목데이터 추가

### 3. 언론사별 기사들을 확인하는 영역
#### (0) 헤더 탭
- [ ] 헤더 탭 레이아웃
- [ ] 내가 구독한 언론사 개수 불러오기 기능
- [ ] 내가 구독한 언론사 뉴스 필터링 기능
- [ ] 그리드/리스트 보기 전환 기능
#### (1) 그리드 보기
- [ ] 가로 930px, 세로 388px에 6*4 테이블
- [ ] 구독하기 버튼 셀 레이아웃
- [ ] 브랜드 마크 셀 레이아웃
- [ ] 브랜드 마크 셀 호버 시 구독하기 버튼 보임 기능
- [ ] 브랜드 마크 순서 새로고침마다 랜덤 배치
- [ ] 그리드 좌우의 화살표로 페이지 넘김 (Disabled 시 표시하지 않음, 최대 4페이지)
#### (2) 리스트 보기
- [ ] 카테고리 영역 레이아웃
- [ ] 카테고리 탭 전환 기능
- [ ] 카테고리 뉴스 필터링 기능
- [ ] 메인 뉴스 레이아웃 (언론사, 편집일, 구독하기 버튼, 이미지, 제목)
- [ ] 뉴스 리스트 요소 레이아웃
